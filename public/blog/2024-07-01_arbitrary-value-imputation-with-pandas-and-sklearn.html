Arbitrary Value Imputation with pandas and sklearn

Handling missing data is a crucial step in any data preprocessing pipeline. One common technique is arbitrary value imputation, where missing values are replaced with a fixed value. This article will guide you through the process of arbitrary value imputation using pandas and sklearn with a practical example based on the Titanic dataset.

Understanding the Data

First, let’s load and inspect the Titanic dataset. This dataset contains information about passengers, such as their age, fare, family size, and whether they survived.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.model_selection import train_test_split
from sklearn.impute import SimpleImputer
from sklearn.compose import ColumnTransformer

# Load the dataset
df = pd.read_csv('titanic_toy.csv')
df.head()

Let’s check for missing values:

df.isnull().mean()

Splitting the Data

We split the dataset into training and testing sets:

X = df.drop(columns=['Survived'])
y = df['Survived']
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=2)

Arbitrary Value Imputation with pandas

We can fill missing values with arbitrary values using the fillna method in pandas. Here, we fill missing Age values with 99 and -1, and missing Fare values with 999 and -1:

X_train['Age_99'] = X_train['Age'].fillna(99)
X_train['Age_minus1'] = X_train['Age'].fillna(-1)

X_train['Fare_999'] = X_train['Fare'].fillna(999)
X_train['Fare_minus1'] = X_train['Fare'].fillna(-1)

Analyzing the Impact on Variance

Replacing missing values can significantly alter the distribution of the data. Let’s compare the variance before and after imputation:

print('Original Age variable variance: ', X_train['Age'].var())
print('Age Variance after 99 wala imputation: ', X_train['Age_99'].var())
print('Age Variance after -1 wala imputation: ', X_train['Age_minus1'].var())

print('Original Fare variable variance: ', X_train['Fare'].var())
print('Fare Variance after 999 wala imputation: ', X_train['Fare_999'].var())
print('Fare Variance after -1 wala imputation: ', X_train['Fare_minus1'].var())

Visualizing the Distribution

We can visualize the effect of imputation on the distribution of the variables using KDE plots:

fig = plt.figure()
ax = fig.add_subplot(111)

# Original Age variable distribution
X_train['Age'].plot(kind='kde', ax=ax)

# Age after 99 imputation
X_train['Age_99'].plot(kind='kde', ax=ax, color='red')

# Age after -1 imputation

Covariance and Correlation

X_train.cov()

Arbitrary Value Imputation with sklearn

We can also perform arbitrary value imputation using sklearn’s SimpleImputer. This is especially useful when integrating the imputation step into a pipeline:

imputer1 = SimpleImputer(strategy='constant', fill_value=99)
imputer2 = SimpleImputer(strategy='constant', fill_value=999)

trf = ColumnTransformer([
    ('imputer1', imputer1, ['Age']),
    ('imputer2', imputer2, ['Fare'])
], remainder='passthrough')

trf.fit(X_train)
X_train = trf.transform(X_train)
X_test = trf.transform(X_test)

Conclusion

Arbitrary value imputation is a simple and effective technique for handling missing data. However, it can significantly impact the distribution and variance of your data, which may affect your model’s performance. Always analyze the effect of imputation on your data and consider multiple imputation strategies to find the best approach for your specific problem.

By understanding and applying these techniques, you can ensure your data preprocessing pipeline is robust and ready for machine learning modeling.
