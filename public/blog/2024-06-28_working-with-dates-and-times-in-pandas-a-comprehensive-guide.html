Working with Dates and Times in Pandas: A Comprehensive Guide

Handling dates and times is a common task in data analysis and can be efficiently managed using the pandas library in Python. This article provides a step-by-step guide to working with dates and times, using practical code snippets to demonstrate how to convert, extract, and manipulate date and time data.

Importing Libraries and Loading Data

First, let’s import the necessary libraries and load the data:

import numpy as np
import pandas as pd

# Loading data
date = pd.read_csv('orders.csv')
time = pd.read_csv('messages.csv')

# Displaying the first few rows of each DataFrame
print(date.head())
print(time.head())

Initial Data Inspection

Before we start manipulating the data, let’s inspect it:

date.info()
time.info()

The date DataFrame contains four columns: date, product_id, city_id, and orders, while the time DataFrame contains two columns: date and msg. Both date columns are currently of the object datatype.

Converting to Datetime

To perform date and time operations, we need to convert the date columns to datetime objects:

# Converting date columns to datetime
date['date'] = pd.to_datetime(date['date'])
time['date'] = pd.to_datetime(time['date'])

# Checking the updated data types
print(date.info())
print(time.info())

Extracting Date Components

Extracting Year, Month, and Day

We can extract specific components of the date, such as the year, month, and day:

# Extracting year, month, and day
date['date_year'] = date['date'].dt.year
date['date_month_no'] = date['date'].dt.month
date['date_day'] = date['date'].dt.day

# Displaying the updated DataFrame
print(date.head())

Extracting Month and Day Names

We can also extract the month and day names:

# Extracting month and day names
date['date_month_name'] = date['date'].dt.month_name()
date['date_dow_name'] = date['date'].dt.day_name()

# Displaying the updated DataFrame
print(date.head())

Extracting Day of the Week and Week Number

Let’s extract the day of the week (as a number) and the week number:

# Extracting day of the week and week number
date['date_dow'] = date['date'].dt.dayofweek
date['date_week'] = date['date'].dt.isocalendar().week

# Displaying the updated DataFrame
print(date.head())

Extracting Quarter and Semester

Finally, we can extract the quarter and semester:

# Extracting quarter and semester
date['quarter'] = date['date'].dt.quarter
date['semester'] = np.where(date['quarter'].isin([1, 2]), 1, 2)

# Displaying the updated DataFrame
print(date.head())

Checking for Weekends

We can create a new column to indicate whether a date falls on a weekend:

# Checking if the date is a weekend
date['date_is_weekend'] = np.where(date['date_dow_name'].isin(['Sunday', 'Saturday']), 1, 0)

# Displaying the updated DataFrame
print(date.head())

Calculating Time Differences

We can calculate the time elapsed between the dates in our data and the current date:

import datetime

# Current date and time
today = datetime.datetime.today()

# Calculating the time difference
date['days_elapsed'] = (today - date['date']).dt.days
date['months_elapsed'] = np.round((today - date['date']) / np.timedelta64(1, 'M'), 0)

# Displaying the updated DataFrame
print(date.head())

Extracting Time Components

Similarly, we can extract time components from the time DataFrame:

# Extracting hour, minute, and second
time['hour'] = time['date'].dt.hour
time['minute'] = time['date'].dt.minute
time['second'] = time['date'].dt.second
time['time'] = time['date'].dt.time

# Displaying the updated DataFrame
print(time.head())

Calculating Time Differences in Seconds, Minutes, and Hours

We can calculate the time differences in seconds, minutes, and hours:

# Time difference in seconds, minutes, and hours
time['seconds_elapsed'] = (today - time['date']) / np.timedelta64(1, 's')
time['minutes_elapsed'] = (today - time['date']) / np.timedelta64(1, 'm')
time['hours_elapsed'] = (today - time['date']) / np.timedelta64(1, 'h')

# Displaying the updated DataFrame
print(time.head())

Conclusion

This guide covers the essential techniques for working with dates and times in pandas. By converting date columns to datetime objects and extracting specific components, you can perform detailed time-based analyses and calculations, which are crucial for many data science and business analytics tasks.
