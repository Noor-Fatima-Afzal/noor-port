Mean and Median Imputation in Pandas: A Practical Guide

Handling missing data is a critical step in any data analysis or machine learning pipeline. One common approach to dealing with missing values is imputation, which involves filling in the missing data with estimated values. Two popular imputation methods are mean and median imputation. In this article, we’ll explore how to perform mean and median imputation using pandas, and we’ll also compare the impact of these methods on the dataset.

Understanding the Dataset

We’ll use a toy Titanic dataset for this demonstration. The dataset includes the following columns:

  • Age: Age of the passengers
  • Fare: Fare paid by the passengers
  • Family: Number of family members accompanying the passengers
  • Survived: Survival status of the passengers (target variable)

Let’s start by loading the dataset and examining its structure.

import pandas as pd

# Load the dataset
df = pd.read_csv('titanic_toy.csv')

# Display the first few rows of the dataset
print(df.head())

Splitting the Data

We’ll split the dataset into training and testing sets to ensure that our imputation strategy is applied only to the training set, avoiding data leakage.

from sklearn.model_selection import train_test_split

# Split the data into features and target
X = df.drop(columns=['Survived'])
y = df['Survived']

# Split the data into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=2)

# Display the shapes of the training and testing sets
print(X_train.shape, X_test.shape)

# Check the percentage of missing values in the training set
print(X_train.isnull().mean())

Imputing Missing Values

Mean Imputation

Mean imputation involves replacing missing values with the mean of the column. Here’s how to perform mean imputation using pandas:

# Calculate the mean of the 'Age' and 'Fare' columns
mean_age = X_train['Age'].mean()
mean_fare = X_train['Fare'].mean()

# Fill missing values with the mean
X_train['Age_mean'] = X_train['Age'].fillna(mean_age)
X_train['Fare_mean'] = X_train['Fare'].fillna(mean_fare)

Median Imputation

Median imputation involves replacing missing values with the median of the column. Here’s how to perform median imputation using pandas:

# Calculate the median of the 'Age' and 'Fare' columns
median_age = X_train['Age'].median()
median_fare = X_train['Fare'].median()

# Fill missing values with the median
X_train['Age_median'] = X_train['Age'].fillna(median_age)
X_train['Fare_median'] = X_train['Fare'].fillna(median_fare)

Comparing Imputation Methods

Let’s compare the variance and distribution of the original and imputed columns to understand the impact of mean and median imputation.

Variance Comparison

# Calculate and print the variance
print('Original Age variable variance: ', X_train['Age'].var())
print('Age Variance after median imputation: ', X_train['Age_median'].var())
print('Age Variance after mean imputation: ', X_train['Age_mean'].var())

print('Original Fare variable variance: ', X_train['Fare'].var())
print('Fare Variance after median imputation: ', X_train['Fare_median'].var())
print('Fare Variance after mean imputation: ', X_train['Fare_mean'].var())

Conclusion

Mean and median imputation are straightforward and effective methods for handling missing data. Mean imputation can be influenced by outliers, while median imputation is more robust in the presence of outliers. By comparing the variance and distribution of the imputed columns, we can make an informed decision about which imputation method to use for a given dataset.

By following the steps outlined in this article, you can effectively handle missing values in your dataset using pandas, ensuring a robust data analysis or machine learning pipeline.
