Detailed Guide to loc and iloc in Pandas

Pandas, a powerful data manipulation library in Python, provides several methods to select and filter data from DataFrames. Two of the most commonly used methods are loc and iloc. This article will delve into the details of these two methods, their slicing behavior, their differences in handling boolean indexing, and how to create new rows and columns using loc.

Understanding loc

The loc method is label-based, meaning you select data by the labels of rows and columns. It is highly flexible and allows for the selection of data based on row and column labels.

Basic Usage

Here is a simple example of how to use loc:

import pandas as pd

# Sample DataFrame
data = {
    'A': [1, 2, 3, 4],
    'B': [5, 6, 7, 8],
    'C': [9, 10, 11, 12]
}
df = pd.DataFrame(data)

# Selecting a single row by label
row = df.loc[0]

# Selecting multiple rows by label
rows = df.loc[[0, 1, 3]]

# Selecting rows and specific columns
subset = df.loc[0:2, ['A', 'C']]

Slicing Behavior with loc

When using slicing with loc, the endpoint is inclusive. This means that when you specify a range, both the start and end labels are included in the output.

# Inclusive slicing with loc
inclusive_slice = df.loc[0:2]  # Includes rows 0, 1, and 2

Creating Rows and Columns with loc

The loc method can also be used to create new rows and columns in a DataFrame. This feature is particularly useful for data manipulation and augmentation.

Adding a New Column

# Adding a new column using loc
df.loc[:, 'D'] = [13, 14, 15, 16]

Adding a New Row

# Adding a new row using loc
df.loc[4] = [17, 18, 19, 20]

Understanding iloc

The iloc method is integer position-based, meaning you select data by the integer locations of rows and columns. It strictly requires integer indices and is useful for selections based on numerical indices.

Basic Usage

Here is an example of how to use iloc:

# Selecting a single row by integer position
row = df.iloc[0]

# Selecting multiple rows by integer positions
rows = df.iloc[[0, 1, 3]]

# Selecting rows and specific columns by integer positions
subset = df.iloc[0:2, [0, 2]]

Slicing Behavior with iloc

When using slicing with iloc, the endpoint is exclusive. This means that the specified range will include the start index but exclude the end index.

# Exclusive slicing with iloc
exclusive_slice = df.iloc[0:2]  # Includes rows 0 and 1, but not 2

Comparison of Slicing Behavior

  • loc[]: Slicing is inclusive of the endpoint.
  • iloc[]: Slicing is exclusive of the endpoint.

Handling Boolean Indexing

Boolean Indexing with loc

The loc method allows for boolean indexing, enabling the selection of data based on conditions. You can use boolean arrays or masks to filter data.

# Boolean indexing with loc
boolean_indexing = df.loc[df['A'] > 2]

Boolean Indexing with iloc

The iloc method does not support boolean indexing directly. It strictly uses integer positions. To achieve similar results, you must first convert the boolean mask to integer positions.

# Boolean indexing with iloc (indirectly)
mask = df['A'] > 2
positions = mask[mask].index
boolean_indexing_iloc = df.iloc[positions]

Summary

  • loc[]: Label-based, inclusive slicing, supports boolean indexing, allows for adding new rows and columns.
  • iloc[]: Integer position-based, exclusive slicing, does not support boolean indexing directly.

Understanding the differences and appropriate use cases for loc and iloc will significantly enhance your data manipulation skills in Pandas. Whether you need label-based selection or integer position-based selection, these methods provide the flexibility and power to handle a wide range of data tasks efficiently.
