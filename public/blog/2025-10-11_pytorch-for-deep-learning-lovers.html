PyTorch for Deep Learning Lovers

Introduction

PyTorch has become one of the most popular deep learning frameworks among researchers and practitioners. Its simplicity, flexibility, and dynamic computation graph have made it the go-to choice for building and training neural networks. Whether you’re a beginner exploring the world of deep learning or an experienced practitioner, PyTorch offers an intuitive interface that makes implementing complex neural architectures straightforward.

In this article, we’ll explore the fundamentals of PyTorch, starting from basic tensor operations and progressing to building complete neural networks.

What is PyTorch?

PyTorch is an open-source machine learning library developed by Meta (formerly Facebook). It provides two key features:

  1. Tensors: Numpy-like arrays that can run on GPUs
  2. Automatic Differentiation: A built-in system to compute gradients automatically

The beauty of PyTorch lies in its dynamic computational graph, which is constructed on-the-fly as operations are executed. This is fundamentally different from static graph frameworks and makes debugging and experimentation much easier.

Getting Started with Tensors

Tensors are the fundamental building blocks of PyTorch. Think of them as multi-dimensional arrays.

Creating Tensors

import torch

# Create a tensor from a Python list
tensor1 = torch.tensor([1, 2, 3, 4])
print("Tensor from list:", tensor1)
# Output: tensor([1, 2, 3, 4])
# Create a 2D tensor (matrix)
tensor2 = torch.tensor([[1, 2], [3, 4], [5, 6]])
print("2D Tensor:\n", tensor2)
# Output:
# 2D Tensor:
#  tensor([[1, 2],
#         [3, 4],
#         [5, 6]])
# Create tensors with random values
random_tensor = torch.randn(3, 3)
print("Random tensor shape:", random_tensor.shape)
# Output: Random tensor shape: torch.Size([3, 3])
# Create a tensor of zeros
zeros = torch.zeros(2, 3)
print("Zeros tensor:\n", zeros)
# Output:
# Zeros tensor:
#  tensor([[0., 0., 0.],
#         [0., 0., 0.]])
# Create a tensor of ones
ones = torch.ones(2, 3)
print("Ones tensor:\n", ones)
# Output:
# Ones tensor:
#  tensor([[1., 1., 1.],
#         [1., 1., 1.]])

Understanding Gradients and Autograd

One of PyTorch’s most powerful features is automatic differentiation (autograd). This computes gradients automatically, which is essential for training neural networks.

Computing Gradients

# Create a tensor with requires_grad=True to track computations
x = torch.tensor([2.0, 3.0], requires_grad=True)
print("x:", x)
# Output: x: tensor([2., 3.], requires_grad=True)

# Perform operations
y = x ** 2
z = y.sum()
print("z:", z)
# Output: z: tensor(13., grad_fn=<SumBackward0>)
# Compute gradients
z.backward()
print("Gradient of x:", x.grad)
# Output: Gradient of x: tensor([4., 6.])
# Explanation: 
# z = x^2 + y^2 = 2^2 + 3^2
# dz/dx = 2x = [4, 6]

Building a Neural Network

Now let’s create our first neural network using PyTorch’s nn module.

Simple Neural Network

import torch
import torch.nn as nn
import torch.optim as optim

# Define a simple feedforward neural network
class SimpleNet(nn.Module):
    def __init__(self):
        super(SimpleNet, self).__init__()
        # Define layers
        self.fc1 = nn.Linear(784, 128)  # Input layer: 28x28=784, Hidden: 128
        self.fc2 = nn.Linear(128, 64)   # Hidden: 128 -> 64
        self.fc3 = nn.Linear(64, 10)    # Output: 64 -> 10 (for 10 classes)
        self.relu = nn.ReLU()
        
    def forward(self, x):
        # Flatten the input
        x = x.view(x.size(0), -1)
        # Pass through layers with activation functions
        x = self.relu(self.fc1(x))
        x = self.relu(self.fc2(x))
        x = self.fc3(x)
        return x
# Create an instance of the network
model = SimpleNet()
print("Model architecture:")
print(model)
# Output:
# SimpleNet(
#   (fc1): Linear(in_features=784, out_features=128, bias=True)
#   (fc2): Linear(in_features=128, out_features=64, bias=True)
#   (fc3): Linear(in_features=64, out_features=10, bias=True)
#   (relu): ReLU()
# )
# Create dummy input (batch_size=1, channels=1, height=28, width=28)
dummy_input = torch.randn(1, 1, 28, 28)
output = model(dummy_input)
print("Output shape:", output.shape)
# Output: Output shape: torch.Size([1, 10])

Training a Neural Network

Training involves forward pass, computing loss, backward pass, and updating weights.

Training Loop

import torch
import torch.nn as nn
import torch.optim as optim

# Create model
model = SimpleNet()
# Define loss function and optimizer
criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)
# Create dummy dataset
X_train = torch.randn(100, 1, 28, 28)  # 100 samples
y_train = torch.randint(0, 10, (100,))  # Random labels 0-9
# Training loop for 3 epochs
num_epochs = 3
batch_size = 10
for epoch in range(num_epochs):
    total_loss = 0
    num_batches = 0
    
    # Iterate through batches
    for i in range(0, len(X_train), batch_size):
        # Get batch
        batch_X = X_train[i:i+batch_size]
        batch_y = y_train[i:i+batch_size]
        
        # Forward pass
        outputs = model(batch_X)
        loss = criterion(outputs, batch_y)
        
        # Backward pass
        optimizer.zero_grad()  # Clear gradients
        loss.backward()        # Compute gradients
        optimizer.step()       # Update weights
        
        total_loss += loss.item()
        num_batches += 1
    
    avg_loss = total_loss / num_batches
    print(f"Epoch {epoch+1}/{num_epochs}, Loss: {avg_loss:.4f}")
# Output:
# Epoch 1/3, Loss: 2.3142
# Epoch 2/3, Loss: 2.2967
# Epoch 3/3, Loss: 2.2801

GPU Acceleration

PyTorch makes it easy to leverage GPU acceleration for faster computations.

Moving to GPU

# Check if GPU is available
print("CUDA available:", torch.cuda.is_available())

if torch.cuda.is_available():
    # Move tensors to GPU
    device = torch.device("cuda")
    tensor_cpu = torch.randn(3, 3)
    tensor_gpu = tensor_cpu.to(device)
    
    print("Tensor on GPU:", tensor_gpu.device)

    # Move model to GPU
    model.to(device)
    print("Model moved to GPU")

Working with Datasets

PyTorch provides utilities for handling datasets efficiently.

from torch.utils.data import Dataset, DataLoader
import numpy as np

# Define a custom dataset
class CustomDataset(Dataset):
    def __init__(self, X, y):
        self.X = X
        self.y = y
    
    def __len__(self):
        return len(self.X)
    
    def __getitem__(self, idx):
        return self.X[idx], self.y[idx]
# Create dataset and dataloader
X = torch.randn(1000, 20)
y = torch.randint(0, 2, (1000,))
dataset = CustomDataset(X, y)
dataloader = DataLoader(dataset, batch_size=32, shuffle=True)
# Iterate through batches
for batch_X, batch_y in dataloader:
    print(f"Batch shape: {batch_X.shape}, Labels shape: {batch_y.shape}")
    break  # Just show first batch
# Output: Batch shape: torch.Size([32, 20]), Labels shape: torch.Size([32])

Practical Tips for Deep Learning with PyTorch

  1. Use DataLoader: Always use DataLoader for efficient batch processing
  2. Move data to device: Remember to move both model and data to GPU
  3. Use model.train() and model.eval(): These set dropout and batch normalization to appropriate modes
  4. Save checkpoints: Save model weights during training to recover best models
  5. Use descriptive variable names: Make your code readable and maintainable
