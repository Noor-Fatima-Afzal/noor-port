🌳Binary Search Tree (BST): A Complete Guide

📘 Introduction

A Binary Search Tree (BST) is a binary tree where each node follows the BST property:

  • All nodes in the left subtree of a node have values less than the node.
  • All nodes in the right subtree of a node have values greater than the node.

This structure enables efficient searching, insertion, and deletion, typically in O(log n) time for balanced trees.

🔁 Traversals in BST

1. In-order Traversal (Left → Root → Right)

  • Produces sorted values in ascending order.
def inorder(node):
    if node:
        inorder(node.left)
        print(node.val, end=' ')
        inorder(node.right)

2. Pre-order Traversal (Root → Left → Right)

  • Useful for copying the tree or expressing in prefix notation.
def preorder(node):
    if node:
        print(node.val, end=' ')
        preorder(node.left)
        preorder(node.right)

3. Post-order Traversal (Left → Right → Root)

  • Often used for deletion (freeing memory).
def postorder(node):
    if node:
        postorder(node.left)
        postorder(node.right)
        print(node.val, end=' ')

🔍 Minimum and Maximum Values

  • Smallest Value = Leftmost node
  • Largest Value = Rightmost node
def find_min(node):
    while node.left:
        node = node.left
    return node

def find_max(node):
    while node.right:
        node = node.right
    return node

🔄 Predecessor and Successor in BST

📌 Definitions:

  • Predecessor: The largest value smaller than a given node.
  • Successor: The smallest value larger than a given node.

General Rules:

  • If node has left child, predecessor is rightmost node of left subtree.
  • If node has right child, successor is leftmost node of right subtree.

✅ Predecessor Code:

def get_predecessor(root, node):
    if node.left:
        return find_max(node.left)

  pred = None
    while root:
        if node.val > root.val:
            pred = root
            root = root.right
        elif node.val < root.val:
            root = root.left
        else:
            break
    return pred

✅ Successor Code:

def get_successor(root, node):
    if node.right:
        return find_min(node.right)

  succ = None
      while root:
          if node.val < root.val:
              succ = root
              root = root.left
          elif node.val > root.val:
              root = root.right
          else:
              break
      return succ

🌐 Tree Traversal: DFS and BFS

🌲 Depth First Search (DFS)

  • Uses a stack (can be recursive).
  • Variants: In-order, Pre-order, Post-order.

🌊 Breadth First Search (BFS)

  • Uses a queue.
  • Visits level by level.
from collections import deque

def bfs(root):
    queue = deque([root])
    while queue:
        node = queue.popleft()
        print(node.val, end=' ')
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)

❌ Deletion in BST (All Cases)

🔹 Three Cases:

  1. Leaf Node (no children) → delete directly.
  2. One Child → replace node with its child.
  3. Two Children → find successor, replace value, then delete successor.
def delete_node(root, key):
    if not root:
        return None
    if key < root.val:
        root.left = delete_node(root.left, key)
    elif key > root.val:
        root.right = delete_node(root.right, key)
    else:
        # Case 1 & 2
        if not root.left:
            return root.right
        elif not root.right:
            return root.left
        # Case 3: Two children
        succ = find_min(root.right)
        root.val = succ.val
        root.right = delete_node(root.right, succ.val)
    return root
