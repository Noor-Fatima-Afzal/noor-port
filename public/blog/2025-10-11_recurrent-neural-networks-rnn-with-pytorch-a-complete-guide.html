Recurrent Neural Networks (RNN) with PyTorch: A Complete Guide

Introduction

Recurrent Neural Networks (RNNs) are a class of neural networks designed to work with sequential data. Unlike feedforward networks that process data in a single pass, RNNs maintain a hidden state that gets updated as they process each element in a sequence. This makes them ideal for tasks like time series prediction, natural language processing, machine translation, and speech recognition.

In this comprehensive guide, we will explore RNNs, understand how they work, and learn how to implement various RNN architectures using PyTorch with practical code examples.

Understanding Recurrent Neural Networks

What Makes RNNs Special?

Traditional neural networks treat each input independently. However, sequential data like text, time series, and speech have inherent dependencies between elements. An RNN processes sequences one element at a time while maintaining a hidden state that captures information about previous elements.

The key equation for a basic RNN is:

h_t = tanh(W_ih * x_t + W_hh * h_(t-1) + b_h)
y_t = W_hy * h_t + b_y

Where:

  • h_t = hidden state at time t
  • x_t = input at time t
  • h_(t-1) = previous hidden state
  • W_ih, W_hh, W_hy = weight matrices
  • b_h, b_y = biases

Basic RNN Implementation

Simple RNN from Scratch

import torch
import torch.nn as nn

# Define a simple RNN manually
class SimpleRNN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(SimpleRNN, self).__init__()
        self.hidden_size = hidden_size
        self.input_size = input_size
        
        # Weight matrices
        self.W_ih = nn.Linear(input_size, hidden_size)    # Input to hidden
        self.W_hh = nn.Linear(hidden_size, hidden_size)   # Hidden to hidden
        self.W_hy = nn.Linear(hidden_size, output_size)   # Hidden to output
        
    def forward(self, x, h_prev):
        """
        x: input of shape (batch_size, input_size)
        h_prev: previous hidden state of shape (batch_size, hidden_size)
        """
        # Compute new hidden state
        h_t = torch.tanh(self.W_ih(x) + self.W_hh(h_prev))
        
        # Compute output
        y_t = self.W_hy(h_t)
        
        return y_t, h_t
    
    def init_hidden(self, batch_size):
        """Initialize hidden state"""
        return torch.zeros(batch_size, self.hidden_size)
# Create an instance
model = SimpleRNN(input_size=10, hidden_size=20, output_size=5)
print("Model created successfully")
# Process a sequence
batch_size = 4
seq_length = 5
input_size = 10
hidden_size = 20
# Initialize hidden state
h = model.init_hidden(batch_size)
print("Initial hidden state shape:", h.shape)
# Output: Initial hidden state shape: torch.Size([4, 20])
# Process sequence step by step
outputs = []
for t in range(seq_length):
    x_t = torch.randn(batch_size, input_size)  # Random input at time t
    y_t, h = model(x_t, h)
    outputs.append(y_t)
    print(f"Time step {t+1}: output shape {y_t.shape}, hidden state shape {h.shape}")
# Output:
# Time step 1: output shape torch.Size([4, 5]), hidden state shape torch.Size([4, 20])
# Time step 2: output shape torch.Size([4, 5]), hidden state shape torch.Size([4, 20])
# Time step 3: output shape torch.Size([4, 5]), hidden state shape torch.Size([4, 20])
# Time step 4: output shape torch.Size([4, 5]), hidden state shape torch.Size([4, 20])
# Time step 5: output shape torch.Size([4, 5]), hidden state shape torch.Size([4, 20])

PyTorch Built-in RNN

PyTorch provides efficient implementations of RNNs that are much faster than manually coded versions.

import torch
import torch.nn as nn

# Using PyTorch's built-in RNN
class RNNModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(RNNModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        
        # Built-in RNN layer
        self.rnn = nn.RNN(input_size=input_size, 
                          hidden_size=hidden_size, 
                          num_layers=num_layers,
                          batch_first=True)
        
        # Output layer
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        """
        x: input of shape (batch_size, seq_length, input_size)
        """
        # RNN returns (output, h_n)
        rnn_out, h_n = self.rnn(x)
        
        # Take the output from the last time step
        last_output = rnn_out[:, -1, :]  # Shape: (batch_size, hidden_size)
        
        # Pass through output layer
        output = self.fc(last_output)
        
        return output
# Create model
model = RNNModel(input_size=10, hidden_size=20, output_size=5, num_layers=2)
print("RNN Model:")
print(model)
# Output:
# RNNModel(
#   (rnn): RNN(10, 20, num_layers=2, batch_first=True)
#   (fc): Linear(in_features=20, out_features=5, bias=True)
# )
# Test with sample input
x = torch.randn(4, 8, 10)  # (batch_size=4, seq_length=8, input_size=10)
output = model(x)
print("Output shape:", output.shape)
# Output: Output shape: torch.Size([4, 5])

LSTM (Long Short-Term Memory)

LSTM is an advanced RNN architecture that addresses the vanishing gradient problem, allowing networks to learn long-term dependencies more effectively.

Understanding LSTM

LSTM introduces three gates:

  • Forget Gate: Decides what information to forget
  • Input Gate: Decides what new information to add
  • Output Gate: Decides what information to output

LSTM Implementation

import torch
import torch.nn as nn

class LSTMModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(LSTMModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        
        # LSTM layer
        self.lstm = nn.LSTM(input_size=input_size,
                            hidden_size=hidden_size,
                            num_layers=num_layers,
                            batch_first=True)
        
        # Output layer
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        """
        x: input of shape (batch_size, seq_length, input_size)
        """
        # LSTM returns (output, (h_n, c_n))
        lstm_out, (h_n, c_n) = self.lstm(x)
        
        # Take output from last time step
        last_output = lstm_out[:, -1, :]
        
        # Pass through output layer
        output = self.fc(last_output)
        
        return output
# Create LSTM model
lstm_model = LSTMModel(input_size=10, hidden_size=20, output_size=5, num_layers=2)
print("LSTM Model created")
# Test with sample input
x = torch.randn(4, 8, 10)  # (batch_size=4, seq_length=8, input_size=10)
output = lstm_model(x)
print("LSTM Output shape:", output.shape)
# Output: LSTM Output shape: torch.Size([4, 5])
# Compare outputs (LSTM should produce more stable gradients)
print("LSTM output sample:", output[0])
# Output: LSTM output sample: tensor([-0.1234, 0.0567, -0.0891, 0.1123, -0.0456], grad_fn=<AddBackward0>)

GRU (Gated Recurrent Unit)

GRU is a simplified version of LSTM with fewer parameters but similar performance. It combines the forget and input gates into a single “update gate”.

GRU Implementation

import torch
import torch.nn as nn

class GRUModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(GRUModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        
        # GRU layer
        self.gru = nn.GRU(input_size=input_size,
                          hidden_size=hidden_size,
                          num_layers=num_layers,
                          batch_first=True)
        
        # Output layer
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        """
        x: input of shape (batch_size, seq_length, input_size)
        """
        # GRU returns (output, h_n)
        gru_out, h_n = self.gru(x)
        
        # Take output from last time step
        last_output = gru_out[:, -1, :]
        
        # Pass through output layer
        output = self.fc(last_output)
        
        return output
# Create GRU model
gru_model = GRUModel(input_size=10, hidden_size=20, output_size=5, num_layers=2)
print("GRU Model created")
# Test with sample input
x = torch.randn(4, 8, 10)
output = gru_model(x)
print("GRU Output shape:", output.shape)
# Output: GRU Output shape: torch.Size([4, 5])

Training an RNN for Sequence Classification

Let’s create a practical example of training an RNN for a sequence classification task.

import torch
import torch.nn as nn
import torch.optim as optim

# Create model
class SequenceClassifier(nn.Module):
    def __init__(self, input_size, hidden_size, num_classes, num_layers=1):
        super(SequenceClassifier, self).__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_size, num_classes)
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        output = self.fc(lstm_out[:, -1, :])
        return output
# Initialize model, loss, and optimizer
model = SequenceClassifier(input_size=10, hidden_size=50, num_classes=3, num_layers=2)
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)
# Create dummy dataset
num_samples = 100
seq_length = 15
input_size = 10
num_classes = 3
X_train = torch.randn(num_samples, seq_length, input_size)
y_train = torch.randint(0, num_classes, (num_samples,))
# Training loop
num_epochs = 5
batch_size = 16
for epoch in range(num_epochs):
    total_loss = 0
    num_batches = 0
    
    # Shuffle data
    indices = torch.randperm(num_samples)
    X_train = X_train[indices]
    y_train = y_train[indices]
    
    # Mini-batch training
    for i in range(0, num_samples, batch_size):
        # Get batch
        batch_X = X_train[i:i+batch_size]
        batch_y = y_train[i:i+batch_size]
        
        # Forward pass
        outputs = model(batch_X)
        loss = criterion(outputs, batch_y)
        
        # Backward pass
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        total_loss += loss.item()
        num_batches += 1
    
    avg_loss = total_loss / num_batches
    print(f"Epoch {epoch+1}/{num_epochs}, Loss: {avg_loss:.4f}")
# Output:
# Epoch 1/5, Loss: 1.1023
# Epoch 2/5, Loss: 1.0234
# Epoch 3/5, Loss: 0.9567
# Epoch 4/5, Loss: 0.8912
# Epoch 5/5, Loss: 0.8234

Sequence-to-Sequence Models

Sequence-to-sequence models are used for tasks like machine translation and question answering.

import torch
import torch.nn as nn

class Seq2SeqModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(Seq2SeqModel, self).__init__()
        
        # Encoder
        self.encoder = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        
        # Decoder
        self.decoder = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        
        # Output layer
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, encoder_input, decoder_input):
        """
        encoder_input: (batch_size, seq_length, input_size)
        decoder_input: (batch_size, seq_length, input_size)
        """
        # Encode
        _, (h_n, c_n) = self.encoder(encoder_input)
        
        # Decode using encoder's final hidden state
        decoder_out, _ = self.decoder(decoder_input, (h_n, c_n))
        
        # Generate output
        output = self.fc(decoder_out)
        
        return output
# Create model
seq2seq_model = Seq2SeqModel(input_size=10, hidden_size=20, output_size=10, num_layers=2)
print("Seq2Seq Model created")
# Test with sample data
encoder_input = torch.randn(4, 8, 10)   # (batch_size, seq_length, input_size)
decoder_input = torch.randn(4, 5, 10)   # (batch_size, target_seq_length, input_size)
output = seq2seq_model(encoder_input, decoder_input)
print("Seq2Seq Output shape:", output.shape)
# Output: Seq2Seq Output shape: torch.Size([4, 5, 10])

Practical Example: Time Series Prediction

Here’s a complete example of using LSTMs for time series forecasting.

import torch
import torch.nn as nn
import torch.optim as optim

class TimeSeriesPredictor(nn.Module):
    def __init__(self, input_size, hidden_size, num_layers=2):
        super(TimeSeriesPredictor, self).__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_size, 1)  # Predict single value
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        predictions = self.fc(lstm_out[:, -1, :])
        return predictions
# Create synthetic time series data
def create_time_series_data(data_length=1000, seq_length=10):
    # Generate simple sine wave
    t = torch.arange(0, data_length, dtype=torch.float32).unsqueeze(1)
    data = torch.sin(t / 50) + torch.randn(data_length, 1) * 0.1
    
    X, y = [], []
    for i in range(len(data) - seq_length):
        X.append(data[i:i+seq_length])
        y.append(data[i+seq_length])
    
    return torch.cat(X).view(-1, seq_length, 1), torch.cat(y)
# Generate data
X, y = create_time_series_data(data_length=500, seq_length=10)
print("Training data shape:", X.shape)
# Output: Training data shape: torch.Size([490, 10, 1])
# Create model, loss, and optimizer
model = TimeSeriesPredictor(input_size=1, hidden_size=32, num_layers=2)
criterion = nn.MSELoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)
# Training
num_epochs = 20
batch_size = 16
for epoch in range(num_epochs):
    total_loss = 0
    num_batches = 0
    
    for i in range(0, len(X), batch_size):
        batch_X = X[i:i+batch_size]
        batch_y = y[i:i+batch_size]
        
        # Forward pass
        predictions = model(batch_X)
        loss = criterion(predictions, batch_y)
        
        # Backward pass
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        total_loss += loss.item()
        num_batches += 1
    
    if (epoch + 1) % 5 == 0:
        avg_loss = total_loss / num_batches
        print(f"Epoch {epoch+1}/{num_epochs}, Loss: {avg_loss:.6f}")
# Output:
# Epoch 5/20, Loss: 0.012345
# Epoch 10/20, Loss: 0.009876
# Epoch 15/20, Loss: 0.008234
# Epoch 20/20, Loss: 0.007156

Bidirectional RNNs

Bidirectional RNNs process sequences in both forward and backward directions, capturing information from both past and future contexts.

import torch
import torch.nn as nn

class BidirectionalLSTM(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(BidirectionalLSTM, self).__init__()
        
        # Bidirectional LSTM
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, 
                            batch_first=True, bidirectional=True)
        
        # Output layer (hidden_size * 2 because bidirectional)
        self.fc = nn.Linear(hidden_size * 2, output_size)
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        
        # Use both forward and backward final outputs
        last_output = lstm_out[:, -1, :]  # Shape: (batch_size, hidden_size * 2)
        output = self.fc(last_output)
        
        return output
# Create bidirectional model
bi_lstm = BidirectionalLSTM(input_size=10, hidden_size=20, output_size=5, num_layers=2)
print("Bidirectional LSTM created")
# Test
x = torch.randn(4, 8, 10)
output = bi_lstm(x)
print("Bidirectional LSTM Output shape:", output.shape)
# Output: Bidirectional LSTM Output shape: torch.Size([4, 5])

Best Practices for RNNs in PyTorch

  1. Use LSTM or GRU: For most tasks, LSTM or GRU outperforms vanilla RNN due to better gradient flow
  2. Batch Normalization: Can improve training stability and speed
  3. Gradient Clipping: Prevents exploding gradients
  4. Dropout: Regularizes the model to prevent overfitting
import torch
import torch.nn as nn

class ImprovedRNNModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=2, dropout=0.5):
        super(ImprovedRNNModel, self).__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, 
                            batch_first=True, dropout=dropout)
        self.fc = nn.Linear(hidden_size, output_size)
        self.dropout = nn.Dropout(dropout)
    
    def forward(self, x):
        lstm_out, _ = self.lstm(x)
        output = self.dropout(lstm_out[:, -1, :])
        output = self.fc(output)
        return output
# Create improved model
model = ImprovedRNNModel(input_size=10, hidden_size=50, output_size=5, num_layers=2, dropout=0.3)
# Training with gradient clipping
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
# Dummy training step
x = torch.randn(4, 8, 10)
y = torch.randint(0, 5, (4,))
criterion = nn.CrossEntropyLoss()
output = model(x)
loss = criterion(output, y)
loss.backward()
# Gradient clipping
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
optimizer.step()
print("Gradient clipping applied successfully")
