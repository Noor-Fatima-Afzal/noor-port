Outlier Detection and Removal Using Z-score Method

Outliers can significantly affect the performance of machine learning models. Detecting and treating outliers is crucial to ensure the accuracy and reliability of the models. This article demonstrates how to detect and remove outliers using the Z-score method with a step-by-step example using a dataset.

Dataset Overview

We will use a sample dataset called placement.csv, which contains three columns: cgpa, placement_exam_marks, and placed.

Visualizing Data Distribution

Let’s visualize the distribution of cgpa and placement_exam_marks.

plt.figure(figsize=(16, 5))

# Calculating Z-score for 'cgpa'
df['cgpa_zscore'] = (df['cgpa'] - df['cgpa'].mean()) / df['cgpa'].std()
print(df.head())

Identifying Outliers

We will identify data points with a Z-score greater than 3 or less than -3.

# Identifying outliers
outliers = df[(df['cgpa_zscore'] > 3) | (df['cgpa_zscore'] < -3)]
print(outliers)

Removing Outliers (Trimming)

We can remove the outliers by filtering out the data points with a Z-score greater than 3 or less than -3.

# Trimming 
new_df = df[(df['cgpa_zscore'] < 3) & (df['cgpa_zscore'] > -3)]
new_df

Capping Outliers

# Capping outliers
upper_limit = df['cgpa'].mean() + 3 * df['cgpa'].std()
lower_limit = df['cgpa'].mean() - 3 * df['cgpa'].std()

df['cgpa'] = np.where(
    df['cgpa'] > upper_limit, 
    upper_limit, 
    np.where(df['cgpa'] < lower_limit, lower_limit, df['cgpa'])
)

print(df['cgpa'].describe())

Conclusion

Outliers can significantly impact the performance of machine learning models. Detecting and treating outliers using methods like Z-score, trimming, and capping ensures that the models built on the data are robust and reliable. By understanding and handling outliers effectively, we can improve the accuracy and generalization ability of our models.
